## Constraints for **Git**

1. Do not commit or push unless explicitly asked.
2. Must follow `commitlint.config.js` conventions and use English.
3. Show the commit message to the user for confirmation first.

## Constraints for **Code**

1. Do not use different names (variables, methods, parameters, etc.) for the same concept.

## Constraints for **Comments**

1. Code comments must be written in English only.
